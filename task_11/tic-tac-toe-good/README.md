# tic-tac-toe-good

A skeleton Go project for the professional Go practices exercise.

## Structure

This project uses a **flat package layout** — domain logic lives in the root package, with `cmd/tictactoe/` as the thin entry point.

## Getting started

```bash
make ci       # lint + test + build
make test     # run tests only
make build    # build the binary to bin/tictactoe
```
